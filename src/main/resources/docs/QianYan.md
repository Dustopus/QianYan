# 📚 千言 AI 助手知识库文档

### Q. 项目基本信息

A：
- **项目名称**：千言
- **英文名称**：QianYan
- **项目类型**：企业级 Agent 系统
- **核心理念**：让沟通更智能，让知识更流动
- **目标用户**：企业内部团队、开发者、知识工作者
- **开发状态**：[开发中]

### Q. 项目背景与愿景

A：「千言」诞生于对高效、智能、安全的企业知识管理与工作流自动化的需求。在信息爆炸的时代，我们希望打造一个基于 LangChain4j 的智能体系统，帮助企业解决知识孤岛与工作流自动化程度低的问题。
"千言"取自成语"千言万语"，寓意即使千言万语，也能在此被 AI 理解与运用。


### Q：这个软件叫什么名字？

A：本软件名为「千言」，寓意"千言万语，尽在其中"。

### Q：千言的核心功能有哪些？

A：千言的核心功能包括：
- **RAG 动态知识植入**：基于 PgVector 的向量检索系统，支持私有领域知识精准检索
- **智能邮箱发送**：通过 Function Calling 实现智能邮件发送
- **联网搜索**：通过 MCP 协议接入联网搜索能力
- **流式输出**：支持 SSE 流式响应，提升交互体验
- **全链路监控**：集成 Prometheus + Grafana 监控体系

### Q：千言使用了哪些核心技术？

A：千言基于以下核心技术栈构建：
- **后端框架**：Spring Boot 3.x
- **AI 框架**：LangChain4j
- **向量数据库**：PgVector (PostgreSQL)
- **对话记忆**：Redis 分布式存储
- **AI 模型**：阿里通义千问 (DashScope)
- **监控体系**：Micrometer + Prometheus + Grafana
